# MAMOGame-Public

Public assets and hosted resources for **MAMOGame** — a mobile game developed by [Yannick Triqueneaux](https://github.com/YannickTriqueneaux) at Haven Studio.

> The game's source code lives in a private repository. This repo contains everything that needs to be publicly accessible: legal pages, static assets, and any future public-facing content.

## Legal Pages

Required by Unity Player Accounts and the Google Play Store.

| Page | URL |
|---|---|
| Terms of Service | [`/tos.html`](https://yannicktriqueneaux.github.io/MAMOGame-Public/tos.html) |
| Privacy Policy | [`/privacy.html`](https://yannicktriqueneaux.github.io/MAMOGame-Public/privacy.html) |

These pages are referenced in:
- Unity Dashboard → Authentication → Unity Player Accounts → Provider Settings
- Google Play Console → Store Listing → Privacy Policy URL
